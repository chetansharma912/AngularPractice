Types of CSS Used in HTML

In HTML, you can use several types of CSS (Cascading Style Sheets) to apply styling. Hereâ€™s a rundown of the main types:

  1. Inline CSS:

    Applied directly within an HTML element using the style attribute.


  2. Internal CSS:

    Defined within a <style> tag inside the <head> section of an HTML document.


  3. External CSS:

    Stored in a separate .css file and linked to the HTML document using the <link> tag.

    Example (styles.css):


  4. Embedded CSS (similar to Internal CSS):

    This is sometimes used interchangeably with internal CSS and involves placing the CSS rules within a <style> tag in the HTML document.

  5. CSS Preprocessors:

    Tools like SASS, LESS, or Stylus extend CSS with features like variables and nested rules. Preprocessed files are compiled into regular CSS files that are then linked in HTML.

  6. CSS-in-JS:

    This approach involves writing CSS directly within JavaScript files, often used with libraries or frameworks like React (using styled-components or emotion). It allows for dynamic styling based on JavaScript logic.

  7. Custom Properties (CSS Variables):

    Defined within CSS using the --variable-name syntax, and used throughout your CSS files


Each type of CSS serves different needs and has its own use cases, with external CSS being the most common for large projects due to its ability to keep styles separate from content.
